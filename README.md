# Snake game

## Index
1. [What is](#What-is)
2. [Control](#Control)
3. [Run](#Run)


### What is
Simple snake game using the Turtle library.  
The game keeps saving the highest score.

### Control
The snake controls it with the directional arrows.  
Press *"q"* to exit.

### Run
```
python3 main.py
```
